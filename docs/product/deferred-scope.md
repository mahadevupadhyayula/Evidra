# Deferred Scope

This document defines what must not be implemented during the MBP unless explicitly re-approved.

## Architecture Deferred

- Next.js.
- FastAPI.
- Separate frontend/backend.
- Redis.
- Celery.
- Distributed workers.
- Microservices.
- pgvector.
- Embeddings.
- Vector search.
- Multi-agent orchestration.
- Workflow engine.
- Model router.
- Prompt dashboard.
- LLM judge.
- Production observability.
- Enterprise multi-tenancy.

## Authentication Deferred

- Google OAuth.
- Magic links.
- SSO.
- Organization accounts.
- Team accounts.
- RBAC beyond single-user ownership.

## Resume/Profile Deferred

- OCR.
- Advanced layout reconstruction.
- Multiple active resumes.
- LinkedIn imports.
- Gmail imports.
- Calendar imports.
- Slack imports.
- Jira imports.
- Notion imports.

## Company/Role Deferred

- Open-ended web search.
- Multi-page crawlers.
- Continuous monitoring.
- News scraping.
- Social scraping.
- Learned role taxonomies.

## Evidence Deferred

- Long-term career graph.
- Multi-source stitching.
- Entity resolution.
- Embedding duplicate detection.
- Auto-merge.

## Stories Deferred

- Advanced version branching.
- Coach-reviewed versions.
- Public publishing.

## Matching Deferred

- Vector matching.
- Learned ranking.
- Interviewer-persona matching.
- Offer probability.

## Payment Deferred

- Subscriptions.
- Tiers.
- Credits.
- Coupons.
- Refund automation.
- Provider-neutral platform.
- Team billing.

## Prep Kit Deferred

- PDF generation.
- Collaborative editing.
- Coach comments.
- Shareable links.

## Practice Deferred

- Audio.
- Video.
- Transcription.
- Filler words.
- Pacing.
- Accent analysis.
- Facial analysis.
- Live interview copilot.

## Plan Deferred

- Notifications.
- Calendar.
- Email reminders.
- Mobile push.
- Habit tracking.

## Admin/Ops Deferred

- Admin recovery UI.
- Full audit logs.
- Data export.
- Deletion workflows.
- Backups.
- Load tests.
- Pentests.
- Cost dashboard.
- Prompt benchmark suite.

A deferred item may be implemented only when the MBP is validated or this document is explicitly updated.
